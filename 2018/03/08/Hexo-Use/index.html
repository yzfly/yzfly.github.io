hexo常用命令

> hexo new post "post_name"   #创建新文章

> hexo new page "page_name"   #创建新页面

> hexo version     #查看版本

> hexo init        #初始化

> hexo clean    #清空缓存

> hexo generate    #收集文件

> hexo server   #本机运行

> hexo depoly   #部署网站

博客写作注意

comments: false #在title出注明此行可关闭评论

关掉每篇博文的伴生文件夹

编辑 _config.yml 文件夹，将下面项设置为false

post_asset_folder: false

使用编辑模板

模板文件在scaffolds文件夹下，编辑post.md文件，以后新建的post文档都使用该模板
