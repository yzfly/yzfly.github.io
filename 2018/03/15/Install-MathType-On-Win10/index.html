在win10上安装并激活MathType
发布于：2018-03-15 10:00:11 标签：/ software / 访问：

MathType是知名的为office办公套件提供公式编辑的软件，但是其为收费软件

在此记录下MathType的安装过程，个人偏爱较为纯净的英文原版软件，需中文版软件请绕道~

软件： MathType-win-en-6.9d（英文原版软件）
博主测试系统： Windows 10 专业版 （64bit）

其他系统应该也能装~自己试试吧~

一、软件下载安装

以下下载方式选择其一即可，如不放心百度云资源请选择官网下载

官网下载：

http://www.dessci.com/en/products/mathtype/

百度云下载

链接：https://pan.baidu.com/s/1UxORrtNuhZVGU_f5dqVqWA 密码：lwdz

下载完成后正常默认安装即可，提示激活时选择30天试用即可

软件激活

在此直接使用注册码激活（无需注册机等，安全可靠）

安装完成后打开MathType，在软件工具栏help中找到 Unlock/Register

其他项随便填，product key一项填入下面激活码其中之一：

序列号：

MTWE691-011016-c8xn5

MTWE691-011772-stfe9

MTWE691-011876-qf14d

MTWE691-011704-e3fyr

MTWE691-011152-33007
