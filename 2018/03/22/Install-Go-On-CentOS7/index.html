Install Go On CentOS7
发布于：2018-03-22 12:48:55 访问：

在CentOS7 64bit上安装golang,流程和安装JDK是一样的，下载安装包，解压到指定安装目录，配置安装路径即可

下载Linux安装包

访问 https://golang.org/dl/ 网站下载相应安装文件

wget golang下载链接
tar -zxvf go.tar.gz #解压安装包，具体文件名按实际而定
mv go /usr/local #将解压后的文件放入 /usr/local/文件夹

配置go安装路径

export PATH=$PATH:/usr/local/go/bin #配置path环境变量
source /etc/profile #使配置立即生效
go version # 查看go的版本
