Use RSA login VPS
发布于：2018-03-09 21:00:13 标签：/ linux环境搭建 / 访问：

近日闲来无事，决定加固一下自己的vps

在此记录下操作过程

目标： 实现RSA指纹登录，禁止密码登录

一、在客户端生成RSA密匙

打开git客户端，在windows上生成RSA密钥

ssh-keygen -t rsa #生成RSA密匙，后面按提示配置
cd ~ #回到用户家目录，生成的密钥在此
cd ./ssh && ls #进入ssh密钥目录
cat id_rsa.pub #输出公钥内容

复制生成的公钥字符串（待会儿要写入服务器配置文件中）

二、将密钥配置到服务器

将上一步公钥写入服务器

vi .ssh/authorized_keys #将密钥复制进去

配置文件权限

chmod 600 .ssh/authorized_keys

三、修改SSH配置文件

开启密钥验证关闭密码登录

vi /etc/ssh/sshd_config

修改以下行

启用密钥验证

RSAAuthentication yes
PubkeyAuthentication yes

指定公钥数据库文件

AuthorsizedKeysFile.ssh/authorized_keys
PasswordAuthentication no

修改ssh登录端口

port 2000 #修改成你想要的端口

重启sshd服务使配置生效

systemctl restart sshd

四、配置xshell登录

1、使用public key验证
2、选择自己的私钥
3、进行登录即可
