Install Git On CentOS7
发布于：2018-03-23 16:25:22 访问：

虽然CentOS提供了yum安装git的方式，然而为了能够使用最新版git我们最好下载源码在本地进行编译安装

安装预编译环境

yum install curl-devel expat-devel gettext-devel openssl-devel zlib-devel -y

下载源码包并解压

cd ~
wget git下载链接 #下载源码包
tar -zxvf git.tar.gz #解压源码包，将解压对象替换成对应包名
cd git #查看解压后生成的文件夹并进入

开始编译安装

cd git #进入解压缩后的文件夹
make prefix=/usr/local all
make prefix=/usr/local install

报错

/usr/bin/perl Makefile.PL PREFIX=’/usr/local’ INSTALL_BASE=’’ —localedir=’/usr/local/share/locale’
BEGIN failed—compilation aborted at Makefile.PL line 3.
　　make[1]: [perl.mak] Error 2
　　make: [perl/perl.mak] Error 2

解决

安装相应的perl包

yum install perl-ExtUtils-MakeMaker package -y
make prefix=/usr/local all && make prefix=/usr/local install #重新编译安装
git —version #查看git版本
