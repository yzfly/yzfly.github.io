Ubuntu上的pip2和pip3
发布于：2018-03-31 09:47:31 访问：

在Ubuntu上如何使python2 和 python3 和谐共处~

20180723更新：最好的办法是安装anaconda创建虚拟环境，使用conda来安装管理Python包，这样可以避免下面的麻烦。

各版本启动命令

python #启动python2
python3 #启动python3

pip2 -V #查看pip2版本
pip -V #查看pip默认版本为2还是3

安装pip3

方法1：

sudo apt install python3-pip -y #安装pip3

sudo apt install python-pip -y #安装pip2，使用pip2 -V查看

pip3 install —upgrade pip %更新pip3
pip3 -V %查看python3 pip版本
pip -V %查看python2 pip版本

pip install —upgrade pip #更新pip

方法2：

进入官网https://pip.pypa.io/en/stable/installing/ 下载get-pip.py文件

python3 get-pip.py #安装pip3
