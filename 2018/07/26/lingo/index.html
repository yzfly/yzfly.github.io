lingo求解优化模型

优化模型中一些通用术语

  • programming: 规划
  • optimization: 优化
  • combinatorial optimization: 组合规划
  • integer programming: 整数规划

优化模型

各种优化模型：

简称 介绍
LP linear programming,线性规划
NLP nonlinear programming,非线性规划
QP quadratic programming,二次规划
IP integer programming, 整数规划，x的一个或者多个分量只取整数值
PIP pure integer programming,纯整数规划，x的所有分量只取整数值
MIP mixed integer programming, x的部分分量只取帧数数值
ZOP zero-one programming, 0-1规划，x的分量中取整数数值的范围限定为只取0或者1
ILP和INLP 整数规划也分为整数线性规划(ILP)和整数非线性规划(INLP)

lingo的语法：

lindo和lingo的区别，lindo主要为较早版本的叫法，lingo为较新版本的命名，两者语法基本相同但有差异，lingo兼容lindo语法。

  1. 注释： ！开头
  2. lingo中假设所有变量非负，所以无需添加非负约束（如 x>0, y>0）

lingo结果解读

lingo状态窗口

名称
Variables:变量
Constraints: 约束变量
Nonzeros: 非零系数约束量
Generator Memory Uesd(K): 内存使用量，单位为千字节
Elapsed Runtime(hh: mm: ss): 求解花费时间，显示格式为“时：分：秒”

lingo Solver Status窗口参数解释

名称 含义
Model Class(模型类型) LP,QP,ILP,IQP,PILP,PIQP,NLP,INLP,PINLP等等（I开头表示IP，PI开头表示)
Status(状态) 显示当前求解状态，“Global optimal”表示找到全局最优解
class(当前模型类型) LP，QP，ILP等
