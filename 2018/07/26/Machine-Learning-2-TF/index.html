超参数

下面列出一些机器学习中常用的超参数：

  • steps: 是指训练迭代的总次数，一步计算一批样本的损失，然后使用该值修改模型的权重一次
  • batch size: 单步的样本数量（随机选择）。
  • periods: 控制报告的粒度。periods的值代表了输出报告的次数。比如学习200次，设置periods=10,则每学习20次输出一次报告
