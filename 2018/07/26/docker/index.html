安装docker

一般情况下，可以直接使用下面命令安装docker，但不推荐使用

这里使用 docker官网提供的方案安装，首先添加官方源，然后安装

1、卸载之前的版本，不会删除容器和镜像等

2、添加docker官方源

3、添加官方 GPG key

4、查看是否添加成功，没有错误输出即可

5、设置 稳定版 源，这里使用的是 ubuntu18.04 系统，若下面命令报错，提示你的发行版本不支持，可以手动修改$(lsb_release -cs) 为你发行版本。比如基于ubuntu18.04的 mint 可以修改为 bionic

6、安装docker

7、将当前用户加入docker组,注销当前会话，重新登录后生效
