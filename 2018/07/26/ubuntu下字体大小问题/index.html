Ubuntu18.04下字体大小问题
发布于：2018-07-26 23:09:57 访问：

在自己笔记本上安装了ubuntu18.04，奈何Ubuntu18.04对高分屏支持不好，字体偏小，gnome桌面可以使用tweak工具设置字体大小，还可以设置屏幕缩放，但是一些软件会因此显示不佳，这里记录下一些软件屏幕字体太小的调整问题。

一、快捷键——最实用

使用 ctrl += 组合键放大 或者使用 ctrl+shift+=三个键

使用 ctrl - 组合键缩小

对大部分软件都可尝试该方法，亲测 Chrome、VScode、SublimeText3等诸多软件都能使用

二、设置

在软件的设置中寻找缩放选项，调节屏幕至自己舒适的程度
