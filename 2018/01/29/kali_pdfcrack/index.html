pdfcrack and fcrackzip
发布于：2018-01-29 23:09:32 访问：

系统环境： kali linux 64bit
在kali linux上利用 pdfcrack工具破解pdf文件密码
利用fcrackzip破解zip文件的弱密码
利用sslscan获取ssl证书信息

pdfcrack

安装pdfcrack

apt-get update -y
apt install pdfcrack -y

开始pdfcrack

此处使用word2016生成的pdf，命名为pdf_name.pdf，设置的密码为123456789

pdfcrack -f pdf_name.pdf -n 6 -m 9 -c 0123456789
对pdf_name.pdf文件进行破解，密码最短为6位，最长为9位，密码字符集为0123456789
开始漫长的等待~

使用密码字典

pdfcrack -f pdf_name.pdf -w passwordlist.txt

fcrackzip

安装fcrackzip(kali自带)

apt-get install fcrackzip -y
man fcrackzip #查看fcrackzip的信息，可忽略

创建测试用zip文件

cd ~
mkdir test
cd test #创建测试用文件夹后进入该文件夹
echo “Hello world” >> hello.txt # 创建测试txt文件
zip —password 123456 hello.zip hello.txt #zip命令先写压缩后文件名，再写要压缩的文件名
设置密码为123456

开始破解

fcrackzip -b -c ‘aA1!’ -l 1-10 -u zip_name.zip #指定使用-b(brute-force)穷举法，指定字符集为大小写字母数字和字符 指定破解文件为zip_name.zip
fcarckzip -D -p /root/password_list.txt -u zip_name.zip
