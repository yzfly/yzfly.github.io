linux上网
发布于：2018-01-29 23:09:32 访问：

系统： kali 64bit

在debian类系统上安装chrome浏览器

安装Chrome

更新源

apt-get update -y && apt-get upgrade -y

下载安装包到本地

下载 chrome linux64.deb安装包到本地，此处假定文件名为 chrome_64bit.deb

开始安装

dpkg -i chrome_64bit.deb
报错，依赖未安装
apt-get install libappindicator1 -y
apt —fix-broken install #修复损坏依赖
dpkg -i chrome_64bit.deb #重新安装

启动

chrome默认不能用root用户启动，解决方法可以创建非root用户启动，或者使用如下命令

/usr/bin/google-chrome —no-sandbox - -user-data-dir &

安装S—

下载文件

cd /root/s~
git clone https://github.com/ssrbackup/shadowsocksr

配置

cp -Rf shadowsocksr /etc/local/
cd /etc/local/shadowsocksr
vi config.json #填好配置
cp config.json /etc/ss.json

本地启动

cd /etc/local/shadowsocksr/shadowsocks/
python local.py -c /etc/ss.json -d #后台运行
