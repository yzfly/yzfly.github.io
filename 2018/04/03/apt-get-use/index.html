APT软件包管理器是Debian平台上的用于安装和删除软件包的命令行工具，通常简写为apt-get.
Debian软件包管理器dpkg可用来维护、升级和安装自定义及第三方程序

APT管理工具

安装软件

apt-get install package_name -y

更新

apt-get update

升级

apt-get upgrade

版本升级

apt-get dist-upgrade

移除软件包

apt-get remove package_name

自动移除

apt-get autoremove

完全移除

apt-get purge package_name

清理

apt-get clean

自动清理

apt-get autoclean

Debian软件包管理器（dpkg）

安装

dpkg -i package_deb.deb

卸载

dpkg -r package_name.deb

完全卸载

dpkg -P package_name.deb

检查已安装软件包

dpkg -l package_name #查看已安装或移除包的状态
dpkg -p package_name #查看已安装软件包的详细信息

Tar命令

顾名思义，tar命令起源于unix系统，它开始是把多个文件写入磁盘文档（Tap ARchive）

创建压缩文件

cd dir_name && tar -cf dir_name*
