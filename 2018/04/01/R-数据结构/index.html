R的学习笔记2

R的对象和属性

R的对象

数值型、整型、字符型、复数型、逻辑型

变量名区分大小写
x <- 3 #原生的赋值方法
x = 3 #此法也可，极端情况下会报错
x <- TRUE #R中逻辑型一定要大写
class(x) #查看x的数值类型

属性

名称、维度、长度（length）、类型（class）

R中的向量（vector）

向量只能包含同一类型的对象

创建向量的三种方法

x <- vector("character",length(10))  #创建字符型，长度为10的空向量
x1 <- 1:4    #创建了1到4的整数型向量
x2 <- c(1,2,3,4)
x3 <- c('a','b','c','d')    #利用c函数创建向量
x4 <- c(TRUE,10,'A')    #R会将不同类型的转换为同一类型，此处为字符型

强制转换

as.numeric()    #强制转换为数值型
as.logical()    #强制转换为逻辑型
as.character()

属性操作

class(x)     #查看变量x的类型
names(x) <- c('name1','name2','name3',)    #为向量x中的每个元素命名

R中的矩阵和数组

矩阵（Matrix）：只能有两个维度 行row和列column

数组（array）：可以有多个维度，矩阵为数组的一种情况

矩阵

x <- matrix(nrow = 3, ncol = 2) # 创建一个三行两列的空矩阵
x <- matrix(1:6,nrow=3,ncol=2)  #创建一个两行三列的空矩阵并赋值
dim(x)    #查看矩阵的维度属性
rbind(x1,y1)  #将x1 y1 两个矩阵进行行拼接
cbind(x2,y2)  #将x2 y2 两个矩阵进行列拼接
attributes(x) #查看矩阵的所有属性

数组

y <- array(1:24,dim=c(4,6))    #创建一个二维数组，也就是矩阵
y2 <- array(1:24,dim=c(2,3,4))    #创建一个维度为三维的数组

R中的列表

列表可以包含不同类型的元素，创建列表的方法为使用list()
l <- list(‘a’,2,10L,3+4i,TRUE) #创建列表
l2 <- list(a=1,b=2,c=3) #为列表中每个元素命名
x <- matrix(1:6,nrow=2,ncol=3) #创建一个矩阵
dimnames <- list(c(‘a’,’b’),c(‘A’,’B’,’C’)) #为矩阵中的行和列命名

R中的因子（factor）

R中的因子用来处理分类数据，区分有序和无序，可以理解为整数向量+标签（优于整数向量）
f <- factor(c(‘female’,’male’,’male’,’female’,’male’)) #创建因子
f2 <- factor(c(‘female’,’male’,’male’,’female’,’male’),levels(‘male’,’female’))

# 使用levels自定义极限水平
table(f)    #查看对f因子的整体描述
unclass(f)    #去掉f的属性
class(unclass(f2))    # 查看去除属性后因子的属性

R中的缺失值

R中的缺失值有两种NA和NaN

NaN属于NA，而NA不属于NaN,NaN常用于判断整数缺失值，而NA使用的范围更广，可以表示多种缺失值，有不同的类型属性

可以使用is.na() is.nan()来判断是否为该类型的缺失值
x <- c(1,2,3,na,na,4)
is.na(x)
is.nan(x)

y <- c(1,2,3,nan,nan,5)
is.na(y)
is.nan(y)

R中的数据框（data frame）

R中的数据框和数据库中的一张表很像，就是数据表，每个元素代表一列数据，元素类型可以不同
d <- data.frame(id=(1,2,3,4),name=c(“a”,”b”,”c”,”d”),gender=c(‘male’,”female”,”male”,”female”)

nrow(d) #查看d的行数
ncol(d) #查看d的列数
data.matrix(dd) #特殊情况下可强制转换为矩阵，前提为数据框中dd中所有元素类型相同

R中的时间与日期

日期：Date

x <- date()                # 读入时间
x2 <- Sys.Date()        #读入系统时间
x3 <- as.Date("2017-3-18")     #存入某一时间，格式为  年-月-日
weekdays(x3)    # 查看x3对应的星期是什么
months(x2)        #查看该时间所属月份
quarters(x3)    #查看该时间所属的季度，一年共有四个季度
julian(x3)        #查看该时间距离1979-01-01过去多少天，转换为一个整数

x4 <- as.Date('2016-01-01')
x5 <- as.Date('2017-01-01')
x5-x4    #日期运算，查看两者相距的天数
as.numeric(x5-x4)  #转换为整数

时间

R中的时间有两种类型（POSIXct、POSIXlt）
t <- Sys.time() #读入系统时间
class(t) #查看时间类型

as.POSIXlt(t) #强制类型转换
as.POSIXct(t) 
unclass(t)    #留下内容去除属性

names(unclass(t))    #查看属性名
t$name1        #取出上一命令中某个名字的具体值

strptime(x1,"%B% %d, %Y %H:%M")  # 读取时间，x1为时间日期字符串，后面%B月份 %d天数 %Y年份 %H小时 %M分钟
