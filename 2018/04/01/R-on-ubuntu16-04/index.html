R on ubuntu16.04
发布于：2018-04-01 16:47:15 访问：

系统环境： ubuntu server 16.04 64 bit
在windows上安装R和Rstudio较容易，下面谈谈如何在ubuntu16.04上安装R和Rstudio server

安装R

可以直接使用如下命令安装R，但是软件源中的R版本较老，为了使用最新版R，我们自己添加R镜像源以获取最新版R

apt-get install r-base 或者 apt-get install r-base-core

移除原来的R

apt-get remove r-base    #删除r-base软件包
apt-get autoremove r-base*

添加最快的镜像

在R官网上找到离自己最快的镜像源，此处选择中科大的镜像
https://mirrors.ustc.edu.cn/CRAN/

vi /etc/apt/sources.list #编辑该文件，将下面镜像源添加到该文件
deb https://mirrors.ustc.edu.cn/CRAN/bin/linux/ubuntu xenial/

安装R

apt-get update -y
apt-get install r-base -y

如果报错，尝试运行如下命令

apt-get install r-base-core -y

检查安装

R —version # 查看R的版本
R #启动R,进入R的交互式shell

安装R包

R #进入R
install.packages(‘txtplot’) #安装txtplot包

安装Rstudio server版

下面贴出官网教程、

https://www.rstudio.com/products/rstudio/download-server/

apt-get install gdebi-core -y
wget https://download2.rstudio.org/rstudio-server-1.1.442-amd64.deb
如果网速过慢可以下载到本机再上传到服务器
gdebi rstudio-server-1.1.442-amd64.deb #若报错可能是安装包损坏，请重新下载该文件

下面是官方使用教程

https://support.rstudio.com/hc/en-us/articles/200552306-Getting-Started

使用Rstudio server,浏览器访问下面的地址

http://server-ip:8787
